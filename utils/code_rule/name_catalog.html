    # 目录命名

    由历史原因及个人习惯引起的 DOM 结构、命名不统一，导致不同成员在维护同一页面时，效率低下，迭代、维护成本极高。

    # 约定

    全部小写，多单词使用半角的连词线 ( - ) 分隔

    • 项目文件夹：projectname
    • 样式文件夹：style
    • 脚本文件夹：js
    • 图片文件夹：img
    • 一般文件夹：array-uniq
